# java-cli-maven-failsafe-spring-findbugs-jacoco-jbehave-mockito-testng-car-data

## Description
A POC for maven app using testng
and jbehave and mockio framework
 with findbugs,
jacoco and failsafe plugin.

## Tech stack
- java
- maven
	- spring
	- mockito
  - testng
  - findbugs
  - jacoco
  - failsafe
  - jbehave

## Docker stack
- adoptopenjdk/maven-openjdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
